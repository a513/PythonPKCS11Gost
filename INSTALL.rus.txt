Установка pyp11
================

Установка из проекта
-----------------------------

  1. Загрузите проект с  https://github.com/a513/PythonPKCS11Gost
  2. Распакуйте проект
  3. Выполните `sudo python3 setup.py install`

Загрузка модуля pyp11 и класса Token
-----------------------------
import pyp11
from Token import Token

-----------------------------
